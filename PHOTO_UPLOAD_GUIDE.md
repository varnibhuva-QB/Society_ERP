## 📝 File Requirements

### Supported Formats
- **Images**: JPG, JPEG, PNG, GIF, WebP
- **Max Size**: 5 MB per image
- **Recommended Resolution**: 1920 × 1440 pixels or higher

### Quality Guidelines
| Aspect | Recommendation |
|--------|-----------------|
| **Format** | JPG for photographs, PNG for graphics |
| **Compression** | Medium compression to balance quality and file size |
| **Aspect Ratio** | 4:3 or 16:9 for best display |
| **Minimum Size** | 640 × 480 pixels |
| **File Size** | Under 3 MB recommended for faster uploads |

### ⚠️ Restrictions
- ❌ Animated GIFs are not supported in some modules
- ❌ BMP and TIFF formats are not accepted
- ❌ Files larger than 5 MB will be rejected
- ❌ Corrupted image files will cause upload failure

## 📍 Upload Locations

### 1. **Member Profile Photos** 👤
- **Location**: Settings → Personal Settings
- **Purpose**: Display member's profile picture
- **Size**: 300 × 300 pixels (square)
- **Recommended Format**: JPG or PNG
- **Maximum Size**: 2 MB

**Steps**:
1. Go to **Settings** (⚙️ icon in sidebar)
2. Click "Edit Profile Photo" or the profile picture placeholder
3. Select image file from your device
4. Crop/adjust as needed
5. Click "Save" or "Upload"

---

### 2. **Property/Flat Photos** 🏠
- **Location**: Members → [Member Details] → Property Photos
- **Purpose**: Documentation of member's flat/property
- **Size**: Multiple photos allowed
- **Recommended Format**: JPG
- **Maximum Size**: 3 MB each

**Steps**:
1. Navigate to **Members** (👥 icon)
2. Select a member
3. Go to **Property Photos** tab
4. Click "Add Photo" button
5. Select multiple images (hold Ctrl to select multiple)
6. Click "Upload"
7. Add captions/descriptions (optional)
8. Confirm upload

---

### 3. **Vehicle Registration Photos** 🚗
- **Location**: All Vehicles → [Vehicle Details] → Documents
- **Purpose**: Vehicle registration certificate, insurance, pollution certificate
- **Recommended Format**: PNG (for documents), JPG (for photos)
- **Maximum Size**: 3 MB each

**Steps**:
1. Navigate to **All Vehicles** (🚗 icon)
2. Click on a vehicle or "Add New Vehicle"
3. Scroll to **Documents** section
4. Click "Upload Document"
5. Select document type (RC, Insurance, Pollution)
6. Upload image/PDF
7. Verify and save

---

### 4. **Notice & Announcement Photos** 📢
- **Location**: Notices → Create Notice
- **Purpose**: Attach images to notices (e.g., event posters)
- **Size**: Multiple images in carousel format
- **Recommended Format**: JPG, PNG
- **Maximum Size**: 4 MB each

**Steps**:
1. Go to **Notices** (🔔 icon)
2. Click "Create New Notice" or edit existing
3. In notice content area, click image icon (🖼️)
4. Select image from device
5. Adjust size/position in editor
6. Continue adding more images if needed
7. Publish/Save notice

---

### 5. **Amenities Gallery** 🏢
- **Location**: Amenities → [Amenity Details] → Gallery
- **Purpose**: Show amenity photos (garden, gym, community hall, etc.)
- **Size**: 4 photos recommended per amenity
- **Recommended Format**: JPG
- **Maximum Size**: 4 MB each

**Steps**:
1. Navigate to **Amenities** (🏢 icon)
2. Select or create amenity
3. Go to **Gallery** tab
4. Click "Add Photo"
5. Upload image
6. Set as cover photo (optional)
7. Save

---

### 6. **Bill & Payment Documents** 📋
- **Location**: Maintenance Billing → Bills → Attachments
- **Purpose**: Attach supporting documents to bills
- **Recommended Format**: PNG (for scans), PDF
- **Maximum Size**: 3 MB each

**Steps**:
1. Go to **Maintenance Bills** (📋 icon)
2. Select bill
3. Click "Attachments" tab
4. Click "Add Attachment"
5. Upload image/document
6. Save bill

## 🔄 Step-by-Step Upload Instructions

### Basic Upload Process

#### Method 1: Direct Click
```
1. Click on image placeholder or "Upload Photo" button
2. Browse and select file from device
3. Wait for upload to complete (progress bar)
4. Confirm/approve if prompted
5. Click Save/Submit
```

#### Method 2: Drag & Drop
```
1. Drag image file from folder
2. Drop onto upload area (usually highlighted)
3. Release mouse button
4. Wait for processing
5. Confirm upload
```

### Upload Progress Indicators
- 🔄 **Blue bar**: File uploading (0-100%)
- ✅ **Green checkmark**: Upload successful
- ❌ **Red X**: Upload failed (see error message)
- ⚠️ **Yellow icon**: Needs manual confirmation

## 🔧 Troubleshooting

### Upload Fails with "File Too Large"
**Solution**:
- Use image compression tool (reduce to under 3 MB)
- Recommended tools: TinyPNG.com, ImageOptimizer.com
- Reduce resolution: Use tools like Photoshop or online converters

### Upload Shows "Unsupported File Format"
**Solution**:
- Check file extension (.jpg, .png, etc.)
- Use online converter to change format (CloudConvert.com)
- Ensure file is actual image (not renamed file)

### Upload Hangs or Gets Stuck
**Solution**:
1. Wait 5-10 seconds for timeout
2. Refresh page (Ctrl+R or Cmd+R)
3. Try uploading smaller file first
4. Clear browser cache
5. Try different browser
6. Contact support if persists

### Image Appears Blurry or Low Quality
**Solution**:
- Use higher resolution source image
- Upload JPG instead of PNG
- Reduce compression when saving
- Check original image quality

### Cannot Access Uploaded Photo
**Solution**:
- Verify you have permissions for that section
- Check if photo was deleted by admin
- Try logging out and back in
- Use different browser
- Contact system administrator

## 🔒 Privacy & Security

### Important Notes
- Only upload photos **with proper permissions**
- Respect privacy of individuals in photos
- Don't share sensitive documents publicly
- Admin can access all uploaded photos
- Photos are stored securely on server
- Delete personal photos when no longer needed

### What Not to Upload
❌ Passwords or confidential information
❌ Financial documents with personal details
❌ Sensitive ID information
❌ Photos of minors without consent
❌ Copyrighted images without permission

## 📞 Support & Help

### Common Questions

**Q: Can I upload from mobile phone?**
A: Yes! Use "Choose File" button on mobile browsers. Some browsers may have different interfaces.

**Q: How many photos can I upload?**
A: Depends on module - typically 5-20 photos allowed. Check specific section limits.

**Q: Can I upload documents (PDF)?**
A: Some modules support PDF (Bills, Documents). Images are universally supported.

**Q: How long are photos stored?**
A: Indefinitely unless manually deleted. Admin can set retention policies.

**Q: Can I download uploaded photos?**
A: Yes, click on photo → "Download" button (if available in your module).
