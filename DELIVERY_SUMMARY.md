## 🎨 Customization Examples

### Change Primary Color (2 min)

1. Edit `theme.js`:
```javascript
colors: {
  primary: '#YOUR_COLOR',
  primaryLight: '#YOUR_LIGHT_COLOR',
  primaryDark: '#YOUR_DARK_COLOR',
}
```

2. Update `tailwind.config.js`:
```javascript
colors: {
  primary: '#YOUR_COLOR',
}
```

### Add Dark Mode (15 min)

```javascript
const [isDark, setIsDark] = useState(false);

<div className={isDark ? 'dark' : 'light'}>
  {/* Content */}
</div>
```

### Change Font Family (5 min)

Update `@apply` in components or add to Tailwind config:
```javascript
theme: {
  fontFamily: {
    sans: ['Your Font', 'fallback'],
  }
}
```
